Mode d'emploi

Le mode d'emploi d'Aseba comprend les sujets introductifs suivants :

  • Démarrer avec Aseba, une introduction pas à pas sur comment programmer avec Aseba, en utilisant le simulateur de robot challenge,
  • Concepts, une description des concepts liés à Aseba,
  • Studio, une description de Studio, l'environnement de développement intégré d'Aseba,
  • Langage, une description détaillée du langage de programmation d'Aseba, avec des exemples.
  • Bibliothèque standard de fonctions natives, une description des fonctions disponibles sur la plupart des cibles Aseba.

Ainsi que les explications avancées suivantes :

  • Remapper les identificateurs avec asebaswitch, une façon de mettre plusieurs robots mono-microcontrôleurs ensemble dans un même réseau aseba.
